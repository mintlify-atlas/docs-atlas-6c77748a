Routes are defined in `src/routes.mjs`. The active path for each route depends on the `usingSEO` flag in `config.json`:

- **`usingSEO: true`** — SEO-friendly names are used (e.g., `/ultraviolet`).
- **`usingSEO: false`** — Alt-paths replace SEO names to obscure keywords that network filters may detect (e.g., `/networking`).

</Note>

## Rammerhead session endpoints

These paths are intercepted by the `serverFactory` before reaching Fastify and are forwarded directly to the in-process Rammerhead server (`src/server.mjs:44`):

| Endpoint | Description |
|----------|-------------|
| `/newsession` | Create a new Rammerhead session |
| `/editsession` | Edit session properties |
| `/deletesession` | Delete a session |
| `/sessionexists` | Check whether a session ID exists |
| `/needpassword` | Check if a session requires a password |
| `/syncLocalStorage` | Sync localStorage state across tabs |
| `/api/shuffleDict` | Retrieve the shuffle dictionary for URL encoding |
| `/mainport` | Return the Rammerhead service port |
| `/messaging` | WebSocket messaging channel |

Any path matching `^/[a-z0-9]{32}` is also routed to Rammerhead as an active session.

## Wisp endpoint

| Endpoint | Protocol | Description |
|----------|----------|-------------|
| `/wisp/` | WebSocket | TCP tunneling for Scramjet and Ultraviolet. HTTP requests to this path are ignored — only WebSocket upgrades are handled. |

## Static asset prefixes

These prefixes are served as static directories and are **not** part of the page routing system:

| Prefix (SEO) | Alt prefix | Content |
|--------------|------------|---------|
| `/assets/` | `/assets/` | CSS, JS, images, JSON |
| `/uv/` | `/network/` | Ultraviolet service worker files |
| `/scram/` | `/worker/` | Scramjet service worker files |
| `/epoxy/` | `/epoch/` | Epoxy transport |
| `/libcurl/` | `/unix/` | libcurl transport |
| `/baremux/` | `/gmt/` | Bare-Mux transport switcher |
| `/chii/` | `/ani/` | Chii remote inspector |
