## Get up and running

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/QuiteAFancyEmerald/Holy-Unblocker.git
    cd Holy-Unblocker
    ```
  </Step>
  <Step title="Install dependencies and build">
    ```bash
    npm run fresh-install
    ```
  </Step>
  <Step title="Start the server">
    ```bash
    npm start
    ```
    The proxy service starts at `http://localhost:8080`.
  </Step>
  <Step title="Access the site">
    Open your browser and navigate to `http://localhost:8080`. Use the omnibox to enter any URL and start browsing through the proxy.
  </Step>
</Steps>

<Note>
  For production self-hosting, switch to a production branch (e.g., `v6.9.5_production`) before deploying. The `master` branch contains work-in-progress changes.
</Note>

## Why Holy Unblocker?

- **No install required** — runs entirely in the browser via service workers
- **Multiple proxy engines** — pick the right engine for your network environment
- **Tor & Onion support** — browse `.onion` sites in any browser, including Chromium
- **Built-in adblocking** — blocks ads across all proxied websites
- **Stealth features** — DOM masquerading and source randomization evade advanced filters
- **30M+ users** — battle-tested since 2020
