## Prerequisites

- **Node.js 20.x** — use [nvm](https://github.com/nvm-sh/nvm) to manage versions
- **Git**
- **NGINX** — required for production deployments (not needed for local development)

<CodeGroup>
```bash nvm (recommended)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
nvm install 20
nvm use 20
```
```bash apt (Ubuntu/Debian)
sudo apt install nodejs npm git
```
</CodeGroup>

## Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/QuiteAFancyEmerald/Holy-Unblocker.git
    cd Holy-Unblocker
    ```
    <Note>
      For production self-hosting, switch to a stable production branch after cloning:
      ```bash
      git checkout v6.9.5_production
      ```
    </Note>
  </Step>
  <Step title="Install dependencies">
    ```bash
    npm run fresh-install
    ```
    This runs `npm cache verify && npm update && npm ci` and also installs Rammerhead's dependencies.
  </Step>
  <Step title="Start the server">
    ```bash
    npm start
    ```
    This stops any running processes, builds static assets, then starts the server. Holy Unblocker will be available at `http://localhost:8080`.
  </Step>
  <Step title="Open in your browser">
    Navigate to `http://localhost:8080`. Type any URL into the omnibox and press Enter to browse through the proxy.
  </Step>
</Steps>

<Warning>
  When `disguiseFiles` is enabled in `config.json` (the default), the entry point is `/login` — not the root URL. Visit `http://localhost:8080/login` first to get a session key, then browsing will work normally.
</Warning>

## Basic usage

Once the server is running, you can:

- **Enter any URL** in the omnibox on the browsing page to proxy it
- **Switch proxy engines** (Scramjet, Ultraviolet, Rammerhead) from the navigation
- **Open Settings** to toggle Tor routing, change transport, or enable/disable adblocking

## Subsequent runs

After the first install, use these commands:

| Command | Description |
|---------|-------------|
| `npm restart` | Stop and restart without rebuilding |
| `npm start` | Full stop + rebuild + start |
| `npm run kill` | Force kill all PM2 processes |
| `npm run clean` | Clear Rammerhead cache |
| `npm run build` | Rebuild static assets only |
| `npm run proxy-validator` | Run endpoint + proxy tests |
