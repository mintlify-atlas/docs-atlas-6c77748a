## What is Holy Unblocker LTS?

Holy Unblocker LTS (Long Term Support) is an open-source, self-hostable web proxy service that bypasses content filters and censorship entirely within your browser. It supports a wide range of blocked sites including YouTube, Discord, Reddit, Instagram, Spotify, and essentially any website — no VPN, no browser extension, no software installation needed.

Over 30 million users have relied on Holy Unblocker since 2020. It runs on three built-in proxy engines — **Scramjet**, **Ultraviolet**, and **Rammerhead** — each suited for different use cases and levels of filter evasion.

## Key features

- **Three proxy engines** — Scramjet, Ultraviolet (UV), and Rammerhead, each with different strengths for bypassing various filter types
- **Tor and Onion routing** — Browse `.onion` sites in any browser, including Chromium, via built-in SOCKS5 proxy support
- **Adblocking** — Built-in adblocking across all proxied websites via a modified service worker
- **DOM masquerading** — Obfuscates real content fetches to evade deep-packet inspection and extension-based filters
- **Source randomization** — Swaps keywords in source files to prevent pattern-based blocking
- **Stealth browsing** — Customize your tab title and icon to avoid detection in browser history
- **Games library** — Integrated HTML5 games, Flash emulation via Ruffle, and RetroArch emulators
- **Production-ready** — PM2 process management, Docker support, and NGINX compatibility out of the box
